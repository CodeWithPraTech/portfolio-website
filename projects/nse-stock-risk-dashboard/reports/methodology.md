# Methodology

## Project Flow

1. Ingest market data.
2. Validate schema, duplicates, date coverage, and OHLC consistency.
3. Build return and risk features.
4. Analyze volatility, drawdown, beta, and correlation.
5. Convert analytics into a dashboard.
6. Add position sizing and decision support after metrics are stable.

## Current App Direction

The live site is intentionally blank while the product direction is being chosen. The project structure is ready for a full dashboard when the scope is finalized.
