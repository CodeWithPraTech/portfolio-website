# Dashboard

The dashboard is the final stakeholder-facing output.

Current views:

- Data health
- Stock list

Completed stock detail:

- SVG candlestick chart from real OHLC rows
- Volume bars
- MA20 and MA50 overlays
- Latest close, 1-day move, 90-session move, and volume-vs-average context

Planned views:

- Risk metrics
- Position sizing
- Decision recommendation

The portfolio site should show each completed capability as it becomes available.
